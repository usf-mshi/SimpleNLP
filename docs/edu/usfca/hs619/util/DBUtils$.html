edu.usfca.hs619.util

DBUtils 

object DBUtils

This scala object is used to handle all the DB related activities. The highlevel apis are driven by the persistent strategy. This objects includes all the DB processing apis and also the high level apis at the same time. Eventually these high-level apis are exposed through SimpleNLP class. In this class the two high level apis are (i) processDischargeSummaries for processing all the summaries. (ii) processDischargeSummaryText for processing a text (iii) getSampleDischargeSummary to get a single discharge summary
